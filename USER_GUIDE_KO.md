# 공동작업 참여자용 사용 가이드

대상:
- 처음으로 `https://antigravity-project-planner.vercel.app/#doc-jsfkkk8` 에 접속해 공동작업을 하는 사용자

## 1. 이 사이트는 무엇을 하는 곳인가?

이 사이트는 여러 사람이 함께 같은 프로젝트 수업 설계 문서를 편집하는 공동작업 도구입니다.

가능한 일:
- 같은 문서를 여러 사람이 함께 작성
- 작성 내용 자동 저장
- 새로고침 후에도 기존 내용 유지
- 같은 링크를 받은 사람끼리 같은 문서에서 협업

## 2. 처음 접속하면 어떻게 해야 하나?

1. 받은 링크를 엽니다.
2. 로그인 안내 화면이 보이면 `이메일 로그인` 버튼을 누릅니다.
3. 본인 이메일 주소를 입력합니다.
4. `로그인 코드 보내기`를 누릅니다.
5. 이메일로 도착한 로그인 코드를 확인합니다.
6. 사이트로 돌아와 코드를 입력합니다.
7. `코드 확인 후 로그인`을 누릅니다.

정상이라면:
- 로그인 직후 기존에 작업하던 문서가 바로 열립니다.

## 3. 로그인 후 무엇이 보이나?

상단에서 확인할 수 있는 것:
- 현재 로그인 상태
- 현재 문서 이름 또는 문서 room
- 공동작업 연결 상태
- 저장 상태

문서 안에서 할 수 있는 것:
- 프로젝트명, 대상 학년, 관련 교과 등 텍스트 입력
- 체크박스 선택
- 차시 수업 설계 내용 작성
- 역할 분담 작성
- 여러 명이 동시에 편집

## 4. 작업 내용은 어떻게 저장되나?

이 사이트는 작성 중인 내용을 자동 저장합니다.

확인 방법:
- 상단의 저장 상태가 `저장 완료`로 보이면 정상입니다.

또한:
- 다른 공동작업자가 같은 문서를 열고 있으면 서로의 변경 내용이 반영됩니다.
- 새로고침해도 저장된 내용이 다시 보입니다.

## 5. 공동작업은 어떻게 하나?

같은 문서에서 함께 작업하려면:
- 같은 링크를 사용해야 합니다.
- 각자 이메일 로그인까지 완료해야 합니다.

공동작업 중에는:
- 같은 문서 내용을 함께 수정할 수 있습니다.
- 다른 사람이 수정한 내용이 반영될 수 있습니다.
- 작업 충돌을 줄이려면 한 항목을 동시에 지나치게 많이 수정하지 않는 것이 좋습니다.

## 6. 문서 이동이나 새 문서 만들기는 어떻게 하나?

상단의 `문서 대시보드`에서 할 수 있습니다.

가능한 작업:
- 최근 작업 문서 다시 열기
- 새 문서 만들기
- 방 이름 또는 코드로 참가하기

주의:
- 공동작업 중인 기존 문서를 계속 써야 한다면, 반드시 받은 원래 링크나 대시보드의 기존 문서를 선택하세요.
- 임의로 새 문서를 만들면 다른 문서로 들어갈 수 있습니다.

## 7. 작업 중 꼭 기억할 점

- 로그인하지 않으면 문서 편집을 시작할 수 없습니다.
- 같은 링크를 받아도 각자 로그인은 따로 해야 합니다.
- 작업 후 새로고침해서 내용이 유지되는지 확인하면 안전합니다.
- 상단 저장 상태가 계속 `저장 중...` 에 머무르면 잠시 기다렸다가 다시 확인하세요.

## 8. 문제가 생기면 먼저 이렇게 해 보세요

### 1) 로그인은 됐는데 문서가 안 보일 때
- 페이지를 한 번 새로고침합니다.
- 그래도 안 되면 링크가 맞는지 확인합니다.

### 2) 이메일 코드가 안 올 때
- 스팸함/정크 메일함을 확인합니다.
- 이메일 주소를 다시 확인하고 코드를 다시 요청합니다.

### 3) 저장이 안 되는 것 같을 때
- 상단 저장 상태를 확인합니다.
- 인터넷 연결 상태를 확인합니다.
- 새로고침 후 기존 내용이 유지되는지 확인합니다.

### 4) 다른 사람과 같은 문서가 안 보일 때
- 같은 링크로 접속했는지 확인합니다.
- 각자 로그인했는지 확인합니다.

## 9. 가장 쉬운 사용 순서

1. 링크 접속
2. 이메일 로그인
3. 기존 문서가 열렸는지 확인
4. 필요한 내용 작성
5. 상단 `저장 완료` 확인
6. 새로고침 후 내용 유지 확인

## 10. 한 줄 요약

이 사이트는 `같은 링크 + 이메일 로그인` 후에 같은 문서를 함께 편집하고 자동 저장하는 공동작업 도구입니다.
